## 🎨 Customization

### Colors
In `css/style.css`:
```css
:root {
    --color-dark-purple: #250e2c;
    --color-purple: #837ab6;
    --color-bright-pink: #f6a5c0;
}
```

### Fonts
In each HTML file's `<head>`:
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;600;700&family=Poppins:wght@300;400;600&display=swap" rel="stylesheet">
```

## 📱 Responsive Design

| Device | Breakpoint | Adjustments |
|---|---|---|
| Desktop | 1200px+ | Full layout |
| Tablet | 768px | Stacked sections, reduced spacing |
| Mobile | 480px | Single column, larger touch targets |
